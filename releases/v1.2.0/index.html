    Release v1.2.0 — Sprint 16: AppSec & Quality¶

    Date: 2026-04-02 Tests: 1549 (1405 → 1549, +144 new)

    Code¶

    Modules Added¶

    • security/hooks.py — Pre-commit config generation (HookEntry, default_hooks, generate/merge YAML)
    • security/quarantine.py — Package quarantine (exclude-newer, PyPI date checking, pin advancement, bypass logging)
    • security/secrets_scanner.py — Secrets scanning (GitleaksBackend, GgshieldBackend, SecretsBackend protocol)
    • security/gate.py — Pre-install SCA gate (5-check battery: pip-audit, GuardDog, OSS health, license, quarantine)
    • security/trivy.py — Trivy multi-ecosystem SCA wrapper (version safety, JSON parsing, compromised version blocking)
    • metrics/baseline.py — Baseline code quality metrics (LOC, test count, commit count, per-module breakdown)

    Bug Fixes¶

    • cli.py — owb security drift default baseline path changed from global (~/.owb/drift-baseline.json) to per-workspace (<workspace>/.owb/drift-baseline.json). The global path caused each workspace to overwrite the previous workspace's baseline.

    Modules Modified¶

    • config.py — SecurityConfig: sca_enabled and sast_enabled defaults changed to True; added trivy_enabled, secrets_scanner, secrets_enabled fields
    • cli.py — Added commands: security secrets, security trivy, security hooks install/status, audit pins, audit gate, metrics baseline; added --all, --secrets, --trivy flags to security scan
    • engine/builder.py — Added _deploy_uv_toml() and _deploy_precommit_config() to build pipeline
    • engine/context.py — Fixed dry-run to check file existence before reporting [write]/[skip]
    • security/scanner.py — Added optional ledger and source params; records FlagEvents for malicious verdicts
    • sources/updater.py — Added reputation ledger check before accepting files; force parameter for override

    Files Added to Repo Root¶

    • .pre-commit-config.yaml — OWB's own pre-commit hooks (gitleaks v8.22.1, ruff v0.11.4)
    • uv.toml — Package quarantine (exclude-newer = "2026-03-25")

    Project Docs¶

    • SDR: Repository structure updated (7 new security modules, metrics package); Sprint 15 and 16 entries added
    • Threat model: 4 new mitigations (M-024 through M-027) for secrets scanning, quarantine, pre-install gate, Trivy

    Content¶

    • vendor/ecc/rules/common/dependency-security.md — Expanded from Python-only to multi-ecosystem (npm, Rust, Go) with quarantine
    • vendor/ecc/rules/common/vault-policies.md — 3 new entries in "When to Consult" table for supply chain
    • content/policies/development-process.md — Supply chain verification added to sprint completion checklist
    • content/policies/product-development-workflow.md — Supply chain gate added to Phase 5 QA
