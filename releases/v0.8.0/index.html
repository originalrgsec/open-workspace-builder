    Date: 2026-03-28 Sprint: 13 Tests: 1253 (1213 → 1253, +40)

    Code Changes¶

    New Modules¶

    • src/open_workspace_builder/tokens/ledger.py — Append-only JSONL ledger with file locking for concurrent hook safety
    • src/open_workspace_builder/tokens/forecast.py — Monthly and sprint cost forecasting with confidence bands
    • src/open_workspace_builder/tokens/budget.py — Budget threshold checking with month-to-date filtering

    Modified Modules¶

    • src/open_workspace_builder/tokens/models.py — Added LedgerEntry frozen dataclass
    • src/open_workspace_builder/config.py — Added TokensConfig (ledger_path, budget_threshold, auto_record)
    • src/open_workspace_builder/cli.py — 5 new commands under owb metrics group
    • src/open_workspace_builder/secrets/age_backend.py — Fixed pyrage identity format (public key comment line)

    New CLI Commands¶

    • owb metrics record — Record session costs to local ledger with --story tagging
    • owb metrics sync — Record + optional Sheets export (sprint-close hook target)
    • owb metrics forecast — Monthly cost projection (text/JSON)
    • owb metrics budget-check — Threshold check with exit code 2 if over budget
    • owb metrics by-story — Cost breakdown by story ID tag
