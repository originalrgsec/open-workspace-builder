## Code Changes

### New Modules
- `src/open_workspace_builder/tokens/` — 7 modules (parser, calculator, models, pricing, reporter, sheets_export, xlsx_export)
- `src/open_workspace_builder/auth/` — Google OAuth flow for Sheets API
- `tests/tokens/` — 82 tests across 6 test files

### New CLI Commands
- `owb metrics tokens` — terminal report with per-project, per-model, per-day breakdowns
- `owb metrics tokens --format json` — machine-readable output
- `owb metrics export --format gsheets --sheet-id <id>` — push to Google Sheets
- `owb metrics export --format xlsx --output <path>` — Excel export
- `owb auth google-store` — encrypt and store OAuth client credentials
- `owb auth google` — run OAuth consent flow for Sheets API

### New Optional Dependencies
- `[sheets]` — google-api-python-client>=2.0, google-auth-oauthlib>=1.0
- `[xlsx]` — xlsxwriter>=3.0
