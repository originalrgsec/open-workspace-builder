# Release v0.4.0 — Sprints 7–8

Released: 2026-03-25

## Stories Completed

| ID | Title | Summary |
|----|-------|---------|
| OWB-S050 | Secrets Backend | Pluggable secrets with keyring, age, and env var backends; `owb auth` CLI; wizard integration |
| OWB-S053 | Dependency Supply Chain Scanning | pip-audit + GuardDog two-layer scanner; `owb audit deps` and `owb audit package` CLI |
| OWB-S055 | Pre-Install SCA Gate | ECC rule instructing agents to run `owb audit package` before pip/uv install |
| OWB-S056 | Semgrep SAST Integration | `owb security sast` CLI wrapping Semgrep with JSON/SARIF output |
| OWB-S057 | SCA/SAST in Evaluator | `--sca` and `--sast` flags on `owb security scan`; trust tier scoring adjustments |
| OWB-S059 | CVE Suppression Monitoring | Suppression registry, `owb audit check-suppressions` CLI, weekly CI job |
| OWB-S064 | Vault Engine Policy Deployment | `VaultBuilder._deploy_policies()` copies content/policies/ to Obsidian/code/ |
| (context) | Context File Lifecycle | ContextDeployer skip-existing, `owb context migrate`, `owb context status` |

## New Modules

### secrets/ package (5 files)

| File | Description |
|------|-------------|
| `secrets/__init__.py` | Package init |
| `secrets/backend.py` | SecretsBackend protocol and get_backend() factory |
| `secrets/keyring_backend.py` | OS keyring backend (macOS Keychain, GNOME Keyring, Windows Credential Manager) |
| `secrets/age_backend.py` | Age encryption backend with pyrage/CLI fallback and auto-keygen |
| `secrets/env_backend.py` | Environment variable fallback backend |

### security/dep_audit.py

Two-layer dependency scanner: Layer A (pip-audit/OSV) and Layer B (GuardDog heuristics). CLI commands, bundled suppressions, CI workflow.

### security/sast.py

Semgrep SAST wrapper with JSON output parsing, SARIF export, configurable rulesets.

### security/data/suppressions.yaml + suppressions_schema.py

CVE suppression registry with validation and OSV API monitoring.

### engine/context.py updates

ContextDeployer with skip-existing detection, interactive migrate, and status reporting.

### engine/vault.py updates

`_deploy_policies()` method for deploying content/policies/*.md to Obsidian/code/.
