# v1.11.0 — Dependency Modernization (Sprint 25)

Released: 2026-04-11

## Summary

Replace OWB's bespoke secrets module with himitsubako and finish genericizing
Claude-specific remnants. Net -2048 lines. Two stories, ~6 story points.

## Code Changes

### Changed

| File | Change |
|------|--------|
| `pyproject.toml` | Add `himitsubako>=0.4.0` core dep; bump `requires-python` to `>=3.12`; rewire `keyring`/`secrets` extras to `himitsubako[keychain]`; remove `age` extra; bump ruff target to `py312` |
| `secrets/__init__.py` | Rewritten as deprecation shim re-exporting from himitsubako (removal: v1.12.0) |
| `secrets/factory.py` | Rewritten to route to himitsubako backends (env, sops, keychain, bitwarden) |
| `secrets/resolver.py` | `_get_backend_name()` helper handles both property and method `backend_name` patterns |
| `config.py` | `SecretsConfig`: removed `age_identity`, `age_secrets_dir`, `onepassword_vault`; added `sops_secrets_file`; removed `ClaudeMdConfig` alias |
| `cli.py` | `auth` commands updated for himitsubako: `auth_backends` uses `shutil.which` + himitsubako imports; `auth_status` simplified; `store-key` uses property-style `backend_name`; `_resolve_age_key_path()` added for Google OAuth |
| `wizard/setup.py` | `_step_secrets_backend()` presents himitsubako's four backends; removed `_check_bitwarden_available()` and `_check_onepassword_available()`; `_step_api_key()` uses property-style `backend_name`; config serialization updated |
| `uv.toml` | `exclude-newer` advanced to `2026-04-12` (first-party exemption for himitsubako) |

### Removed

| File | Reason |
|------|--------|
| `secrets/base.py` | Protocol superseded by `himitsubako.backends.protocol.SecretBackend` |
| `secrets/env_backend.py` | Superseded by `himitsubako.backends.env.EnvBackend` |
| `secrets/keyring_backend.py` | Superseded by `himitsubako.backends.keychain.KeychainBackend` |
| `secrets/age_backend.py` | Superseded by himitsubako's SOPS+age backend |
| `secrets/bitwarden_backend.py` | Superseded by `himitsubako.backends.bitwarden.BitwardenBackend` |
| `secrets/onepassword_backend.py` | No himitsubako equivalent; dropped from OWB |
