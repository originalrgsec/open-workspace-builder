## Architecture Notes

Two new package namespaces added to `src/open_workspace_builder/`:

- **evaluator/** — Skill quality assessment pipeline. Chains classification, test generation, execution, scoring, and judgment into three evaluation modes. Uses OWB's LiteLLM ModelBackend for all LLM operations. Data files in evaluator/data/ provide weight vectors, examples, and policies.

- **sources/** — Multi-source content infrastructure. Replaces the hardcoded ECC update path with a config-driven system supporting arbitrary upstream sources. Each source has its own discovery patterns, audit rules, and update pipeline.
